## Features

- Add New Todo Task 🆕
- Delete Todo 💥
- Delete Confirmation Popup 📃
- Mark Todo as done ➡️
- Count The #Number of Todos Active or Completed **SOON**
- Light/dark mode toggle **SOON**
- Search for Todo -Realtime-
- Inline Edit Todo Task **SOON**
- Fullscreen mode
- Responsive
- Cross platform 💻
- Data Stored in the Browser 🖇️
